# 📱 Настройка мобильного приложения для работы с деплоем

## 🚨 ТЕКУЩИЙ СТАТУС: API НЕ РАБОТАЕТ

**Проблема:** Сервер на `https://linadugau-mystik-39d3.twc1.net` возвращает HTML вместо JSON для всех API endpoints.

**Причина:** На сервере развернута старая версия кода, где статические файлы обрабатываются ДО API роутов.

**Решение:** Нужно пересобрать и перезапустить сервер с новым `production-server.js`.

## 🚀 Как запустить мобильное приложение:

### 1. Установите зависимости
```bash
npm install
# или
bun install
```

### 2. Запустите приложение
```bash
# Для разработки
npx expo start

# Для туннеля (если нужен доступ с физического устройства)
npx expo start --tunnel
```

### 3. Протестируйте API подключение
```bash
# Запустите тест API
node test-mobile-api.js
```

Должен показать:
```
🎉 Все тесты прошли! Мобильное приложение готово к работе.
```

## 🆘 Если что-то не работает:

### 1. Проверьте API сервер
```bash
node test-mobile-api.js
```

### 2. Проверьте консоль Metro
Ищите ошибки сети или API в консоли разработчика

### 3. Проверьте настройки сети
- Убедитесь, что устройство подключено к интернету
- Проверьте, что нет блокировки HTTPS запросов

### 4. Очистите кеш
```bash
npx expo start --clear
```
