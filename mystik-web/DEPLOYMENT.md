## Настройки для деплоя на React 24 платформе

### 🚀 Команды для деплоя:
**Команда сборки:**
```
npm install --legacy-peer-deps && cd mystik-web && npm install --legacy-peer-deps && npm run build:no-check && cd server && npm install --legacy-peer-deps
```

**Команда запуска:**
```
cd mystik-web && npm run start:full
```

### 🔧 Переменные окружения:
```
NODE_ENV=production
PORT=8000
VITE_API_URL=https://linadugau-mystik-39d3.twc1.net/
```

### 📁 Директории:
- **Директория до сборки:** `/mystik-web/dist`
- **Путь до директории проекта:** `/` (корень)
- **Ветка:** `main`
